Шаблоны

Шаблоны на DTL (Django Template Language) используются в Django для более удобной работы с HTML и позволяют использовать в них некоторые элементы программирования вроде циклов.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
    <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">
    <title>Homepage</title>
</head>
<body>
    <div class="container mt-5">
        <h1 class="mb-4">Welcome to "Racing Club" racing club</h1>
        <a href="{% url 'login' %}" class="btn btn-primary mb-2">Log in</a>
        <a href="{% url 'register_racer' %}" class="btn btn-secondary mb-2">Register</a>
    </div>
